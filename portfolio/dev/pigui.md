# pigUI

Bei pigUI handelt es sich um ein UI Management System / Framework für pygame. Es übernimmt die Kontrolle über das Element-Layering, die Renderreihenfolge und es blockiert die Eingabe von anderen UI Elementen sollte bereits eines im Fokus sein. 

Erweiterte Pygame Events für eine intuitive und simplifizierte Eventbehandlung. Zum Beispiel: Doppelklicks, Scrollen und Tastenwiederholung.

Texteingaben, Menüfelder, Dropdowns & Colorpicker - pigUI hat dies alles integriert damit sich der Entwickler nur noch um das Entwickeln der Features kümmern muss.

UI und UX Logik ist strikt getrennt um eine leichtere Wartbarkeit zu gewährleisten, dies ermöglicht einen rapiden Wechsel zwischen verschiedenen Themes.
