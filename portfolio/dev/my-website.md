# Meine Webseite

Mit dem Beginn meiner Weiterbildung in der Masterschool startete ich auch dieses Projekt. Angefangen mit WordPress und minimalistischem HTML durch die gegebenen Limitationen, über eine "graue" bis hin zur heutigen Version, welche inspiriert von Wäldern, Bunkeranlagen und einem Hauch Fantasie mein Portfolio schmückt.

Die WordPress Diskrepanzen wurden vorübergehend mit guten alten HTML, CSS & JavaScript und späterem Umstieg auf Vite, React & TypeScript gelöst, was wohl die beste Entscheidung überhaupt war.

Das Entwickeln mit einer neuen Sprache / neuem Frameworks macht immer gute Laune. Durch mein Praktikum bei [webeet.io](), lerne ich unter anderem genau dies, mehr Kniffe und Tricks.

### Technik

Alle `blogs`, `portfolio-einträge`, `bilder` & ähnliches werden aus dem [Backend GitHub Repository]() gezogen. Ganz simpel über `index.json` und für jeden Eintrag eine Markdowndatei.

Das hält das Projekt klein, macht alles übersichtlicher und einfacher schnelle Änderungen durchzuführen.
