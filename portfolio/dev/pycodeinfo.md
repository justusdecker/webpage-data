# Pycode-Info

Dieses Package umfasst Code-Zusammenfassungen, eine zyklomatische Komplexitätsanzeige und andere Programmiersprachenstatistiken. Mit dem Flake8 Linter werden alle PEP-Richtlinien-Verstöße sofort aufgedeckt.

* Heatmap Visualisierung: Bei Grenzwertüberschreitung schlägt die zKa Alarm. Z.B. bei zu langen Funktionen oder einer zu langen Line im Code.
* Programmiersprachenstatistiken: Eine visualisierte Anzeige, die einem Aufschluss über die unterschiedlichen Programmiersprachen gibt. Es ist eine Zusammenfassung wie bei GitHub, nur im Terminal.
