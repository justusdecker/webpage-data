# WIN

WIN ist ein kleines Wortzähler-Ratespiel, welches Daten von Wikipedia nutzt, gefetched mit meiner eigenen "API".

## Gameplay

Der User muss erraten wie viele Wörter sich in einem Wikipediaeintrag befinden. Drei Optionen stehen zur Auswahl. Rät der User richtig, gibts Punkte, wenn nicht, dann nicht.

Die Entwicklung wurde nach dem Hackathon nicht weitergeführt.
