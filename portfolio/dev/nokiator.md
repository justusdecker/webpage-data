# Nokiator

Nokiator ist ein FFmpeg-basiertes Windows-Batch Tool zum konvertieren von Videos, Audios und Bild um diese vor allem auf älteren Geräten, wie z.B. dem Nokia RM 1190 zuverlässig abspielen zu können.

Nicht nur die Zuverlässigkeit ist ein wichtiger Punkt, sondern auch das durch die optimierte Ausgabe z.B. bei einer Audioaufnahme - Eine "Konstante Bitrate von 128 kbps" beträgt, diese gewährt eine längere Akkulaufzeit, da die Datenmenge "weggeschnitten" wird, die nicht zwingend benötigt wird. 

Das Tool unterstützt Einzel-/Batchverarbeitung für alle gängigen Medienformen(MP4, MP3, ...). Durch simples einstellen der Encoderflags über '%ENCODER_FLAGS%', kann die Ausgabe bei Bedarf sogar noch weiter optimiert werden.
