# 🚀 Start eines neuen Nebenprojekts: Entwicklung einer Lebenslaufvorlage für Entwickler!
Ich habe mit der Entwicklung einer eigenen Lebenslauf-Engine begonnen (Teil meines großen Application-Manager-Projekts), um von Standard-Dokumenteditoren wegzukommen.

![CV](https://media.licdn.com/dms/image/v2/D4E22AQFIsQCV3hQ12w/feedshare-shrink_1280/B4EZrCXxy_KYAs-/0/1764197606807?e=1774483200&v=beta&t=GZYI4Ru2f8jbCThB8S-mbKqrxTI2NKqsGEV2omJ5-pI)

Die aktuelle Version verwendet reines HTML/CSS mit Fokus auf Print-Media-Queries (@media print), um eine gestochen scharfe Darstellung auf PDF und Papier zu gewährleisten.

Es handelt sich derzeit um eine Vorabversion, aber Seitenleiste und Hauptraster funktionieren bereits.

Ich plane außerdem, einen Konverter zu entwickeln, der Daten aus einer JSON-Datei mithilfe von Jinja in das HTML einfügt, sodass ich das reine HTML nie wieder bearbeiten muss.

Das Beste daran: Dieses Projekt enthält eine Ente!🦆
