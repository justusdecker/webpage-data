# Projekt abgeschlossen: Lebenslaufvorlage erstellt und Lebenslauf-Erstellungsprozess automatisiert. 🧾
Ich freue mich, die Fertigstellung meines neuesten Nebenprojekts für den „ApplicationManager“ bekanntzugeben.

* Lebenslauf-Automatisierung: Das Projekt nutzt Jinja, um die Platzhalter automatisch auszufüllen.
* Einfache Bedienung und Anpassung: Sie müssen lediglich eine Datei kopieren, anpassen und ein einfaches Python-Skript ausführen.
* Freie Nutzung: Das Projekt ist lizenzfrei und kann ohne Lizenzprobleme in Ihrem Repository verwendet werden.

👉 Code und weitere Details findet Ihr [hier](https://github.com/justusdecker/CV-Template).
